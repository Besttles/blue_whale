## 命令行操作

---

* 通过客户端连接ZooKeeper的集群

```
/Users/onlyone/software/zookeeper/zookeeper-3.4.8/bin/zkCli.sh -server 192.168.0.24:2181
```

* help打印命令行帮助


* 创建一个节点

```
create /node 'node value'
```

* ls，查看/目录内容

```
[zk: 192.168.0.24:2181(CONNECTED) 3] ls /
[node, zookeeper]
```

```

[zk: 192.168.0.24:2181(CONNECTED) 26] 
```
